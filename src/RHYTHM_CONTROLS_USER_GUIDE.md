## Troubleshooting

### Problem: "No theme to apply rhythm to" error
**Solution:** Create at least one note in your theme first

### Problem: Rhythm sounds wrong
**Solution:** Check analytics to see actual distribution, try a preset

### Problem: Can't hear rhythm differences
**Solution:** 
1. Make sure volume is up
2. Try more contrasting rhythms (sixteenths vs. wholes)
3. Use slower tempo to hear clearly

### Problem: Rhythm reset when adding notes
**Solution:** This is expected - rhythm auto-syncs with theme length. Reapply rhythm after editing theme.
