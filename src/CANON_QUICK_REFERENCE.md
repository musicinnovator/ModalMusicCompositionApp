## ‚öôÔ∏è Parameter Guide

### Essential Parameters (All Canons)
- **Type**: Choose from 14 canon types
- **Interval**: Transposition distance (semitones)
- **Delay**: Entry timing (beats)

### Optional Parameters
- **Mensuration Ratio**: Speed multiplier (1.5 = slower, 0.67 = faster)
- **Inversion Axis**: Center pitch for mirror reflection
- **Num Voices**: Total number of voices (2-8 recommended)

## üöÄ Getting Started

### Step 1: Create a Theme
- Use Theme Composer
- 8-16 notes recommended
- Keep range within 1-2 octaves

### Step 2: Choose Canon Type
- Select from dropdown in Canon Generator
- Each type has different parameters

### Step 3: Adjust Parameters
- **Interval**: Try octave (12) or fifth (7) first
- **Delay**: 4 beats is standard
- **Voices**: 2-3 for clarity, 4+ for richness

### Step 4: Generate & Listen
- Click "Generate Canon"
- Each voice gets its own instrument
- Toggle individual voices on/off
- Adjust individual instruments

### Step 5: Add to Song
- Go to "Complete Song Creation Suite"
- Drag canon from "Available Components"
- Arrange on timeline
- Export as MIDI

## üé® Creative Tips

### For Beginners
- Start with **STRICT_CANON** at octave (12 semitones)
- Use 2-3 voices only
- Keep delay at 4 beats
- Choose complementary instruments (piano + violin)

### For Intermediate
- Try **INVERSION_CANON** for contrary motion
- Experiment with **RHYTHMIC_CANON** (ratio 1.5 or 2.0)
- Use **PER_TONOS** for modulating sequences
- Try 4-5 voices

### For Advanced
- Combine **RETROGRADE_INVERSION_CANON** with **DOUBLE_CANON**
- Create **ENIGMATICUS** with custom transformations
- Use **MENSURABILIS** with complex ratios (3:2, 4:3)
- Layer multiple canons in Song Suite
- Export and analyze in DAW
