# Rhythm Controls - Quick Start Guide

## üéµ Where to Find Rhythm Controls

### 1. **Main Theme**
üìç Location: Theme Composer ‚Üí "Rhythm Controls" section (above Rest Controls)

**Quick Steps:**
1. Add notes to your theme
2. Click on "Rhythm Controls" to expand
3. Toggle between "Individual" or "Automatic" mode
4. Set rhythm values for each note
5. Play to hear your custom rhythm

---

### 2. **Bach Variables**
üìç Location: Theme Composer ‚Üí "Bach Variables" tab ‚Üí Select a variable

**Quick Steps:**
1. Switch to "Bach Variables" tab
2. Select any Bach Variable (CF, FCP1, etc.)
3. Add notes to the variable
4. Use "Rhythm Controls" section below the variable selector
5. Each variable maintains its own rhythm independently

---

### 3. **Imitations** ‚≠ê NEW
üìç Location: Scroll down to any Imitation card ‚Üí Each part has Rhythm Controls

**Quick Steps:**
1. Generate an imitation using "Imitation & Fugue Controls"
2. Scroll to the imitation visualization card
3. Find "Rhythm Controls" below each part melody:
   - **Original Part** - First RhythmControls section
   - **Imitation Part** - Second RhythmControls section
4. Customize rhythm for each part independently
5. Play imitation to hear both rhythms together

---

### 4. **Fugues** ‚≠ê NEW
üìç Location: Scroll down to any Fugue card ‚Üí Each voice has Rhythm Controls

**Quick Steps:**
1. Generate a fugue using "Imitation & Fugue Controls"
2. Scroll to the fugue visualization card
3. Find "Rhythm Controls" below each voice melody:
   - **Voice 1 Rhythm**
   - **Voice 2 Rhythm**
   - **Voice 3 Rhythm**
   - etc.
4. Customize rhythm for each voice independently
5. Play fugue to hear all voices with custom rhythms

---

## üéπ Rhythm Control Modes

### Individual Mode
- Click on each note's rhythm badge
- Set rhythm values independently
- Perfect for complex rhythmic patterns

### Automatic Mode (Theme & Bach Variables only)
- Use percentage slider (0-100%)
- Automatically distributes rhythm values
- Quick way to create rhythmic variation
- 0% = mostly longer notes (whole, half)
- 100% = mostly shorter notes (eighth, sixteenth)

---

## üì§ Exporting with Custom Rhythms

All rhythm data is **automatically preserved** when:

1. **Playing audio** - AudioPlayer uses your custom rhythms
2. **Creating songs** - Song Composer includes all rhythm data
3. **Exporting MIDI** - Correct note durations in MIDI file
4. **Exporting MusicXML** - Proper note values in score notation
5. **Exporting TXT** - Rhythm values listed in text format

## üîç Troubleshooting

**Problem:** Rhythm controls don't appear for my imitation/fugue  
**Solution:** Scroll down to the visualization card - controls are below each melody

**Problem:** Changes to rhythm don't affect playback  
**Solution:** Make sure you're clicking the "Play" button on the specific composition card

**Problem:** Exported MIDI has wrong rhythm  
**Solution:** Add composition to Song Composer first, then export the complete song

**Problem:** Can't find rhythm controls for counterpoint  
**Solution:** Counterpoint rhythm is automatic (from species counterpoint algorithm)

## üí° Tips & Tricks

1. **Create rhythmic counterpoint:** Set different rhythms for each voice in a fugue
2. **Augmentation:** Double all rhythm values for a slower voice
3. **Diminution:** Halve all rhythm values for a faster voice
4. **Syncopation:** Use eighth/sixteenth notes strategically
5. **Rhythmic independence:** Each composition maintains separate rhythm data
