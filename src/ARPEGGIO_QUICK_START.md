# Arpeggio Pattern System - Quick Start Guide
**Get started with arpeggios in 2 minutes!**

---

## ‚ö° Fast Track (30 seconds)

1. **Create a simple theme**: Click "Triad Base" preset (C, E, G)
2. **Find the Arpeggio section**: Scroll down in Theme Composer ‚Üí Traditional tab
3. **Click "Apply Arpeggio Pattern"** (default pattern already selected)
4. **Done!** Your theme is now an arpeggio: C - E - G

## üéØ Common Patterns

### Ascending Arpeggio
```
Pattern: LMH (3 notes)
Source: C, E, G
Result: C ‚Üí E ‚Üí G
```

### Descending Arpeggio
```
Pattern: HML (3 notes)
Source: C, E, G
Result: G ‚Üí E ‚Üí C
```

### Wave Pattern
```
Pattern: LMHML (5 notes)
Source: C, E, G
Result: C ‚Üí E ‚Üí G ‚Üí E ‚Üí C
```

### Double Wave
```
Pattern: LMHLMH (6 notes)
Source: C, E, G
Result: C ‚Üí E ‚Üí G ‚Üí C ‚Üí E ‚Üí G
```

## üìã Step-by-Step

### For Main Theme (Traditional Mode)

1. **Open Theme Composer**
2. **Create a theme** with at least 3 notes:
   - Use "Triad Base" preset, OR
   - Click note buttons (C4, E4, G4), OR
   - Use MIDI keyboard
3. **Scroll to "Arpeggio Pattern Generator"**
4. **Check the note preview**:
   - L (Low) = lowest note in your theme
   - M (Mid) = middle note
   - H (High) = highest note
5. **Select pattern**:
   - Click tab: 3, 4, 5, or 6 notes
   - Choose pattern from dropdown
6. **Adjust repetitions** (optional):
   - Slider: 1-8 times
7. **Click "Apply Arpeggio Pattern"**
8. **Your theme is updated!**

### For Bach Variables (Cantus Firmus, etc.)

1. **Switch to "Bach Variables" tab** in Theme Composer
2. **Select your target variable** (e.g., Cantus Firmus)
3. **Add notes** to that variable
4. **Scroll to "Arpeggio Pattern Generator"**
5. **Select and apply pattern** (same as above)
6. **Variable updated!**

## üí° Quick Tips

### Tip 1: Preview First
Always check the L, M, H preview before applying to see which notes will be used.

### Tip 2: Start Simple
Begin with 3-note patterns, then try more complex ones.

### Tip 3: Use Repetitions
Need a longer theme? Increase repetitions instead of choosing longer patterns.

### Tip 4: Experiment
Try different patterns with the same source theme - results vary dramatically!

### Tip 5: Combine Features
Generated arpeggio + Rhythm Controls + Imitation Engine = powerful compositions!

## ‚ö†Ô∏è Common Mistakes

### ‚ùå Empty Theme
**Problem**: "Source theme is empty" error  
**Solution**: Add at least one note before using arpeggios

### ‚ùå Only One Note
**Problem**: L, M, H all show the same note  
**Solution**: Add notes with different pitches (e.g., C, E, G)

### ‚ùå Pattern Too Short
**Problem**: Generated arpeggio is only 3 notes  
**Solution**: Increase repetitions or choose longer pattern

### ‚ùå Wrong Variable
**Problem**: Arpeggio applied to wrong Bach variable  
**Solution**: Check which tab is selected before applying

## üé® Pattern Types At-a-Glance

### Ascending
- **LMH** - Straight up
- **LMHL** - Up and return
- **LMHLMH** - Double ascent

### Descending
- **HML** - Straight down
- **HMHL** - Down and return
- **HMLHML** - Double descent

### Wave
- **LMHML** - Full wave
- **MLHM** - Wave from middle
- **HLMH** - Wave from high

### Oscillating
- **LHLM** - Low-high jumps
- **MHMH** - Mid-high alternation
